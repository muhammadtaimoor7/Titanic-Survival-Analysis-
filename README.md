# Titanic Survival Analysis

## Aim
To analyze Titanic dataset and find out which 
factors affected the survival of passengers.

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
tested.csv — 418 rows, 12 columns

## Key Findings
- Females survived more than males
- 1st class passengers survived more than 3rd class
- Most passengers were between 20-35 years old
- 3rd class had highest number of deaths
